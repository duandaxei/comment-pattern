# 图案注释插件

在打开的文件上，右键菜单，点击【图案注释】

当然，您也可是通过快捷键触发，默认快捷键如下：

```json
{
    "command": "extension.commentPattern",
    "key": "Ctrl+Shift+/",
    "macosx": "command+Shift+/"
}
```

## 自定义快捷键

HBuilderX 菜单【工具】【自定义快捷键】，可以自定义图案注释快捷键:

```json
{
    "key": "",
    "command": "extension.commentPattern"
}
```
